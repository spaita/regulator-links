# 🚀 Regulator Auto-Link Opener

A one-click bookmarklet that automatically opens all merchant review links in background tabs.

## ⚡ Features

- **Saves ~30 seconds per case**
- Opens 8+ tabs automatically (Business Name, Owner Email, Business Phone, Activation, TLO, TLO2, Radiation, Employee)
- Excludes Negative News
- Fast and reliable

## 🌐 Live Demo

Visit: **https://anticssp.github.io/regulator-bookmarklet/**

## 📊 Time Savings

- Per day (20 cases): ~10 minutes
- Per week: ~50 minutes  
- Per month: ~3.5 hours
