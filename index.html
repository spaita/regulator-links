📥 Installation (Takes 1 Minute)

Show Your Bookmarks Bar

Mac: Press Cmd + Shift + B

Windows: Press Ctrl + Shift + B

Create the Bookmarklet

Right-click anywhere on your bookmarks bar

Click "Add Page..." or "Add Bookmark"

Name: 📂 Open All Links (or whatever you prefer)

Copy & Paste the Code

Click the big green button below to copy the code

Paste it into the URL field of your new bookmark

Click Save

📝 The Code

Important: Make sure to copy the ENTIRE code including javascript: at the beginning!
javascript:(function(){let count=0;const links=document.querySelectorAll('a');links.forEach(link=>{const href=link.href;const text=link.textContent.trim();if(!href||!href.startsWith('http')||href.includes('javascript'))return;const isNegativeNews=text.toLowerCase().includes('embezzle')||href.toLowerCase().includes('embezzle');if(isNegativeNews)return;const isRelevant=(href.includes('google.com/search')||text.match(/TLO|Radiation|Employee|2FA/i)||link.closest('.merchant-info'));if(isRelevant){window.open(href,'_blank');count++;}});setTimeout(()=>alert('✅ Opened '+count+' tabs!'),200);})();

⚠️ Troubleshooting

Nothing happens when I click it

✓ Make sure the code starts with javascript: (no spaces)
✓ Make sure you copied the ENTIRE code
✓ Try in Chrome (works best)

Opens wrong links or too many tabs

✓ Make sure you're on a Regulator merchant review page
✓ You might see both TLO and TLO2 - just close the one you don't need

Alert doesn't show

✓ Check your browser's pop-up blocker settings
✓ The tabs should still open even if alert is blocked

Bookmark disappeared

✓ Re-create it following the steps above
✓ Make sure you clicked "Save" when creating it
